# Lesson 01 - Users

## Creating a New Linux User

In this task, I created a new Linux user and created a home directory for the account.

### Command Used

```bash
sudo useradd -m alxndr
```

### Verification

I verified that the user was created successfully using:

```bash
id alxndr
```
